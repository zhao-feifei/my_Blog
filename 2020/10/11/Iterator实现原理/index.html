迭代器原理剖析

一. 啥是迭代器

迭代器（Iterator）是一种接口，为各种不同的数据结构提供统一的访问机制。任何数据结构只要部署 Iterator 接口，就可以完成遍历操作。ES6创造了一种新的遍历命令for…of循环，Iterator接口主要供for…of消费， 原生具备iterator接口的数据(可用for of遍历)的数据有：

a) Array

b) Arguments

c) Set

d) Map

e) String

f) TypedArray

g) NodeList
